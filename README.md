- `index.html`: dashboard principale

- `data.json`: dataset pulito e analisi
- `style.css`: stile grafico

- `details.js`: logica pagina dettagli

## Pubblicazione rapida
1. Crea un nuovo repository GitHub.
2. Carica tutti i file contenuti in questa cartella.
3. Attiva GitHub Pages dalle impostazioni del repository, pubblicando dal branch principale.
4. Apri l'URL generato da GitHub Pages.
